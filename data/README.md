# Sample Data

Example inputs for testing and demonstration.

---

## sample-inputs/

Contains:
- `sample-takeoff.json` — Example takeoff with quantities and labor hours
- `sample-catalog.csv` — Example product catalog with SKUs, prices, suppliers
- `sample-kits.json` — Example kit mappings (takeoff item → list of SKUs)

These files demonstrate the data formats the tool expects.

---

Use these to:
- Test the tool during development
- Show how the data flows
- Validate calculations
